Projeto de carrinho com RFID.

Esse projeto consiste simplesmente na criação de um carrinho de super mercado automatizado, onde o primeiro passo é criar um dispositivo que a partir do momento que o usuário insere uma mercadoria no carrinho é acrescentado o valor do mesmo, tendo assim no display o valor exato de quanto o usuário vai precisar pagar na hora de chegar no caixa.
